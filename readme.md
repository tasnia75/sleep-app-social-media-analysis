# Social Media Engagement Analysis: Calm vs Headspace

This repository contains a group project completed for the **MKTG5504 Big Data in Marketing** course.

The project analyses how two leading digital wellness brands, **Calm** and **Headspace**, generate engagement on **Facebook and Instagram**. Using R, the project examines how content type, emotional tone, and language style influence user engagement metrics such as likes, comments, shares, and views.

## Project Objective

The aim of this project was to identify the key drivers of engagement across platforms and provide data-driven recommendations for a new sleep app entering the market.

## Repository Structure

- `code/` – R script used for preprocessing, statistical analysis, text analysis, and modelling
- `presentation/` – final presentation slides
- `data/` – note about the dataset
- `outputs/` – folder for generated files and engineered datasets

## Methods

The analysis combines statistical and text-mining techniques to examine engagement patterns across social media platforms.

Key analytical methods include:

- Descriptive statistics
- Correlation analysis
- Chi-square tests
- Independent-sample t-tests
- Regression modelling
- Dictionary-based text analysis

## Text Analysis Dictionaries

The following dictionaries were used to analyse linguistic patterns in social media posts:

- **Regulatory Focus** (promotion vs prevention framing)
- **Regulatory Mode** (locomotion vs assessment language)
- **Brand Personality** (Aaker framework)
- **NRC Emotion Lexicon** (emotion detection)
- **AFINN Sentiment Lexicon** (sentiment scoring)

## Tools & Libraries

The project was implemented in **R** using libraries such as:

- `dplyr`
- `ggplot2`
- `quanteda`
- `tidytext`
- `corrplot`
- `psych`

## Key Findings

- Video posts generated the strongest engagement on both Facebook and Instagram
- Instagram engagement performed well with short, emotionally positive content
- Calm was associated with reflective and sophisticated messaging
- Headspace benefited more from promotional and action-oriented framing
- Emotional tone and content style both influenced likes, comments, and shares
